# IceCube: Neutrinos in Deep Ice

## about
Поиск частиц нейтрино глубоко во льдах Антарктиды. 
Этот проект - baseline-решение соревнования Kaggle, с фокусом на инженерные практики (Polars, MLflow, модульная архитектура).

## stack
* **Обработка данных:** Python, Polars
* **Machine Learning:** LightGBM, Scikit-learn
* **MLOps:** MLflow, DagsHub
* **Конфигурация:** PyYAML
